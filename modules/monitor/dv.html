Reading and Handling of DV objects¶

SeisMIC uses DV objects to handle velocity changes. Those can for example saved, loaded, and plotted.

Read a DV Object from Disk¶

Most likely, you will want to ready the velocity changes that were computed in the previous step. SeisMIC uses binary npz format to store those. You will find the files in the folder that you have defined earlier (i.e., in the yaml file). Load the object with seismic.monitor.dv.read_dv(). It only takes one argument: the path to the dv object.

from seismic.monitor.dv import read_dv

dv = read_dv('/path/to/my/dv/DV-net0-net1.stat0-stat1.ch0-ch1.npz')
